# Mia Launcher

**Mia Launcher** es la aplicación de escritorio oficial del ecosistema [Mia](https://miadsc.xyz): una herramienta para gestionar tus cuentas de Epic Games, consultar información de Fortnite (Battle Royale y Salvar el Mundo) y automatizar tareas que normalmente harías desde el bot de Discord.

> Versión actual publicada en este repositorio: consulta la pestaña [Releases](https://github.com/MyNameIsPako/Mia-Launcher/releases) para ver la última build disponible.

---

## Descarga e instalación

1. Entra en **[Releases](https://github.com/MyNameIsPako/Mia-Launcher/releases)** y abre la versión más reciente.
2. Descarga el instalador de Windows:
   - `Mia-Launcher_*_x64-setup.exe`
3. Ejecuta el instalador y sigue los pasos del asistente.
4. Abre **Mia Launcher** desde el menú de inicio o el acceso directo del escritorio.

También puedes descargar la última versión desde la web oficial: **[miadsc.xyz](https://miadsc.xyz)**

### Requisitos

| Requisito             | Detalle                                                              |
| --------------------- | -------------------------------------------------------------------- |
| **Sistema operativo** | Windows 10/11 (64 bits)                                              |
| **Conexión**          | Internet estable (Epic Games, Discord y API de Mia)                  |
| **Cuenta Discord**    | Obligatoria para iniciar sesión en Mia Launcher                      |
| **Cuenta Epic**       | Necesaria para la mayoría de funciones (añádela tras iniciar sesión) |

> El launcher se actualiza automáticamente cuando hay una nueva versión publicada en este repositorio.

## ¿Qué puedes hacer con Mia Launcher?

### Salvar el Mundo y Battle Royale

- **Alertas de misiones** — Misiones de STW, materiales de evolución, supervivientes, héroes y más.
- **Tienda de BR** — Consulta la tienda actual, filtra objetos y compra o regala con tus cuentas.
- **Estado de servidores** — Comprueba si los servicios de Epic están operativos.
- **Misiones diarias de STW** — Revisa y gestiona tus misiones.
- **Inventario STW** — Mochila, almacén y aventuras, con imágenes, filtros y búsqueda.
- **Taquilla BR** — Todos tus cosméticos de Battle Royale (skins, gestos, picos, etc.) con filtros y valor estimado en paVos.
- **XP ganada** — Progreso semanal en BR, Creativo y STW.
- **Taxi** — Juega misiones de STW por encima de tu nivel de poder.
- **Auto-kick** — Expulsión automática al terminar la misión, reclamar recompensas, transferir materiales e invitar amigos.
- **Estado personalizado** — Cambia tu mensaje de presencia en Fortnite.
- **Buscar jugadores** — Consulta perfiles públicos.
- **Consola MCP** — Peticiones avanzadas al perfil de Fortnite (usuarios experimentados).

### Cuentas y Epic Games

- **Varias cuentas** — Añade y cambia entre cuentas guardadas de forma local.
- **Amigos** — Gestión de lista de amigos y lista blanca global.
- **paVos** — Consulta el saldo de tus cuentas.
- **Código SAC** — Añade, revisa o cambia tu código de creador de apoyo.
- **Canjear códigos** — Activa códigos de Epic en tus cuentas.
- **EULA** — Acepta los acuerdos pendientes de Epic.
- **Sitio web de Epic** — Acceso rápido con sesión de tus cuentas.

### Autenticación (desarrolladores y bots)

- Generar **access tokens**, **exchange codes** y **device auth** para tus cuentas.

### Biblioteca de juegos (Windows)

- **Biblioteca** — Ver juegos instalados de Epic Games.
- **Descargas** — Descargar e iniciar juegos desde la Epic Games Store (Legendary).

## Primera vez: pasos recomendados

1. **Inicia sesión con Discord** (obligatorio) y luego añade tus cuentas de Epic Games.
2. Revisa las **alertas de misiones** en la pantalla principal.
3. En **Ajustes**, activa las opciones que necesites (por ejemplo, logs de depuración si tienes problemas).
4. Explora el menú lateral: las secciones con candado requieren tener al menos una cuenta conectada.

## Apoya a Mia

Si te gusta el proyecto, usa el código de creador de apoyo **`NotMia`** en la Tienda de Fortnite. También puedes configurar tu código SAC desde la sección **Cuenta → SAC** del launcher.
