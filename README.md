# Mia Launcher

App de escritorio oficial de [Mia](https://miadsc.xyz) para gestionar cuentas Epic, consultar Fortnite (BR y STW) y automatizar tareas del bot.

**Descarga:** [Releases](https://github.com/MyNameIsPako/Mia-Launcher/releases) (`Mia-Launcher_*_x64-setup.exe`) o [miadsc.xyz](https://miadsc.xyz)

**Requisitos:** Windows 10/11 (64 bits), internet, cuenta Discord (login) y Epic (mayoría de funciones). Se actualiza solo al publicar una nueva release.

## Funciones principales

- **STW / BR:** alertas de misiones, tienda BR, servidores, misiones diarias, inventario STW, taquilla BR, XP, taxi, auto-kick, estado personalizado, buscar jugadores, consola MCP.
- **Cuentas:** varias cuentas Epic, amigos, paVos, SAC, canjear códigos, EULA, web de Epic, tokens/códigos para desarrolladores.
- **Windows:** biblioteca Epic y descargas (Legendary).

**Idiomas:** ES, EN, DE, FR, PT-BR, TR — en Ajustes.

## Primeros pasos

1. Inicia sesión con **Discord** y añade cuentas Epic.
2. Revisa alertas en inicio; el menú con candado pide cuenta Epic.
3. Código de apoyo: **`NotMia`** (también en Cuenta → SAC).
